1. Select Element with Aria-Label

The select element has aria-label="Pick a color for the task"

2. Label Element

The label contains text: "Pick a color for the task"

3. Multiple Detection Methods

  • aria-label attribute
  • data-testid attribute
  • name attribute (e.g., "taskColor")
  • title attribute

4. How to Find in Browser DevTools

Open DevTools and run:

document.querySelector('select[aria-label*="Pick a color"]')

5. How AI Should Interact

Most Reliable Method:

await page.selectOption('select[aria-label*="Pick a color"]', '#B624FF');

Alternative Methods:

  • await page.selectOption('select[name="taskColor"]', '#B624FF');
  • await page.selectOption('[data-testid="color-picker-select-taskColor"]', '#B624FF');
  • await page.getByRole('combobox', { name: /pick a color/i }).selectOption('#B624FF');
